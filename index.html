Project Description & Unique Value

Arcane Runner aims to combine tight, deterministic player movement and dash mechanics with bullet-hell inspired boss encounters. The project focuses on modular demos for rapid iteration (laser pickup, boss sprites, movement scripts, dash mechanic), visual clarity, and playtestable build packages.

Notable Challenges & Solutions

  • 360° projectile spread issue — resolved by reworking projectile spawn and using deterministic vector math.
  • Projectile direction bug — solved using switch-case direction mapping and normalization.
  • Dash not passing player coords — debugged collision checks and corrected velocity clamping logic.
